#  lol英雄秒选器


#### Build Setup

``` bash
# install dependencies
yarn or yarn install

# serve with hot reload at localhost:9080
yarn dev

# build electron application for production
yarn build


```
---
# 功能

- 自动更新
- 秒选英雄
- 绰号筛选
- 秒选后发话
- 查询队友最近20局战绩（胜率，KDA，常用英雄）
- 自动发队友战绩
- 百度统计
